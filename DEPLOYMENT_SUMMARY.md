## 🚀 使用方法

### 快速开始（推荐新手）
```bash
# 1. 构建项目
npm run build

# 2. 部署到GitHub Pages
npm run deploy:github

# 或使用部署工具
npm run deploy github
```

### 高级部署
```bash
# 检查环境
npm run check

# 部署到Netlify
npm run deploy:netlify

# 部署到Vercel
npm run deploy:vercel

# 部署到Firebase
npm run deploy:firebase
```

### 自动化部署
- 推送代码到main分支自动部署到GitHub Pages
- 支持PR预览功能
- 包含构建状态通知
